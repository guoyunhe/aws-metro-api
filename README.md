# aws-metro-api

## Local Development

### System Requirements

- Node.js
- aws cli https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html
- awscli-local (pip install awscli-local)
- Docker

### Setup

```
docker-compose up
awslocal s3api create-bucket --bucket test
```

## Environments

- production
- staging
- development
